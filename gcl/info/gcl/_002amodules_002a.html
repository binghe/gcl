24.2.10 *modules* [Variable]

Value Type::

a list of strings.

Initial Value::

implementation-dependent.

Description::

The value of *modules* is a list of names of the modules that have been loaded into the current Lisp image.

Affected By::

provide

See Also::

provide , require

Notes::

The variable *modules* is deprecated.
